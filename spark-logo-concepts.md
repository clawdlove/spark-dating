# Spark Logo Concepts

## Direction: Match Head + Spark (Minimalist)

### Concept A — The Strike
```
    ✦  (spark/ignition)
    |
   ( )  ← match head (circle)
    |
   / \  ← match stick
```

Simple, iconic. The spark above the match head suggests the "light" moment.

---

### Concept B — Ignition
```
    ⟿  (curved spark line)
    |
   (•)  ← match head with glow
```

A curved spark line hitting the match head. More dynamic, suggests motion.

---

### Concept C — Single Match
```
   ★
   |
   ○
   |
  ╱ ╲

```

Star spark above a solid match head. Clean and minimal.

---

### Concept D — Flame Trail
```
  ⟲
   \
    (o)
     \
      ‾

```

A curved stroke that becomes the match flame. Very minimal, almost abstract.

---

### Concept E — Phosphorus Glow
```
    ◐
   ╱  ╲
  (    )
   ╲  ╱
    ‾

```

Match head with a half-shadow, like the phosphorus strip on a matchbox. Subtle, sophisticated.

---

All using coral (#FF6B6B) as primary. Want me to refine any of these, or try different variations?
